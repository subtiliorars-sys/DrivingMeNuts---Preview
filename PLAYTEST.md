## 3. First session checklist (~30–45 min)

- [ ] Complete the **tutorial** (buy raw peanuts → open roast slot → end day → read the day report).
- [ ] Play **2–3 more days** — change price at least once, buy stock, run one roast.
- [ ] Note anything **confusing**, **unfun**, or that teaches the **wrong business lesson**.
- [ ] Note **bugs** with numbered steps to reproduce.
- [ ] Optional: try Settings (motion / color options) and browser zoom if text feels small.

## 5. What this game is

A cozy pixel idle/RPG about a **roasted-peanut food truck** that teaches real small-business ideas (pricing, COGS, cash flow) without spreadsheet pain. Running gag: customers explain that peanuts are legumes; the owner is already tired of hearing it.

**Not affiliated** with any recovery program. **Not** financial advice.
