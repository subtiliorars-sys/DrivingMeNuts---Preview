# Driving Me Nuts

*A cozy pixel-art RPG + idle game about running a roasted-peanut food truck.*

> Run a peanut-roasting food truck, learn real small-business skills (margins, cash flow, pricing), and survive the occasional legume lecture.

**Status:** Active development — playable demo ready (version 0.x).

---

## Features

- **Roast, price, and sell peanuts** with real supply-and-demand economics. Set your margins too high and watch customers walk; too low and you'll never cover the cart payment.
- **Build relationships with NPC customers** — they remember how you treated them, and loyal customers unlock new recipes and upgrades.
- **Manage cash flow, debt, and growth** — learn what "COGS" means by feeling it. Take out loans, pay them back (or don't), and discover why every small business needs a rainy-day fund.
- **100% privacy-safe.** All game data lives in your browser's local storage. No accounts, no telemetry, no data leaves your machine.

---

## The Game So Far

Current demo includes: save/load with corruption recovery, daily profit-and-loss ledger, achievement badges, collectible Legume Lore entries, weather that affects customer turnout, an auto-sell upgrade, supplier relationships, brand campaigns, accessibility settings (reduced motion, color-blind support), an in-game business glossary, and a four-path "Save the Truck" rescue arc.

## Building from Source

```bash
git clone https://github.com/subtiliorars-sys/DrivingMeNuts---Preview
cd DrivingMeNuts---Preview
npm install
npm run dev
```

Runs locally at `localhost:5173`.
